# UK Energy Demand Forecasting & Market Analysis

## Project Overview

This project analyses UK electricity demand data and builds machine learning models to forecast national electricity demand using time-series features.

The aim of the project is to understand how electricity demand changes over time, identify daily and weekly demand patterns, and compare different forecasting models against simple baseline approaches.

This project is designed as a portfolio project for data analysis, machine learning, and time-series forecasting roles.

---

## Key Objectives

- Explore UK electricity demand patterns over time
- Analyse daily, weekly, and seasonal variation in demand
- Engineer time-series features such as lag variables and rolling averages
- Build baseline forecasting models for comparison
- Train machine learning models to predict electricity demand
- Evaluate model performance using appropriate regression metrics
- Interpret model results and discuss limitations

---

## Dataset

The project uses UK electricity demand data containing half-hourly observations across a full year.

The dataset includes variables such as:

- National Demand
- Transmission System Demand
- England and Wales Demand
- Embedded wind generation
- Embedded solar generation
- Interconnector flows
- Pump storage demand
- Settlement date and settlement period

The target variable used in the modelling section is national electricity demand.
