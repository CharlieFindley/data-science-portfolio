# Football Match Prediction & Betting Market Analysis

## Project Overview
This project analyses Premier League match data and bookmaker odds to predict football match outcomes. The aim is to compare machine learning models against bookmaker-implied predictions and evaluate whether football statistics, betting market data, or combined features perform best.

## Dataset
The dataset comes from Football-Data.co.uk and contains Premier League match results, match statistics, and bookmaker odds.

## Methods Used
- Data cleaning
- Exploratory data analysis
- Feature engineering
- Bookmaker favourite baseline model
- Logistic regression
- Random forest classifier
- Model comparison across different feature sets

## Key Questions
- How accurate are bookmaker favourites?
- Can machine learning models outperform simple betting-market baselines?
- Are football performance features useful compared with bookmaker odds?
- Does combining football and betting features improve prediction accuracy?

## Files
- `football_project_finished.ipynb` — main project notebook
- `historical_football_data.csv` — dataset used in the project

## Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## Summary
The project compares bookmaker-based predictions, football-statistic-based models, and combined models to assess how well match outcomes can be predicted from historical Premier League data.
